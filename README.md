# Minesweeper
지뢰찾기 게임 만들기

## 설치 및 실행 방법

```
$ npm install 
$ npm run dev
```
## 라이브러리
- react
- redux-toolkit
- typescript
- styled-components

## 구현사항
- [x]  첫번째 빈칸을 열었을 경우에는 지뢰가 터지면 안됩니다.
- [x]  타이머
- [x]  난이도 변경 가능
    - Beginner (8X8), Intermediate (16X16), Expert (32X16)
    - Custom (가로, 세로, 지뢰 수 조정 가능)
- [x]  오른쪽 클릭 깃발, 물음표 기능
